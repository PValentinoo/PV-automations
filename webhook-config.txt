WEBHOOK CONFIGURATION
=====================

To set up the webhook for your contact form, create a .env file in your project root with:

VITE_WEBHOOK_URL=https://your-webhook-url-here.com/webhook

EXAMPLE WEBHOOK SERVICES:

1. Discord Webhook:
   VITE_WEBHOOK_URL=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN

2. Slack Webhook:
   VITE_WEBHOOK_URL=https://hooks.slack.com/services/YOUR_WEBHOOK_URL

3. Custom API Endpoint:
   VITE_WEBHOOK_URL=https://your-api.com/webhook

4. n8n Webhook:
   VITE_WEBHOOK_URL=https://your-n8n-instance.com/webhook/contact-form

5. Zapier Webhook:
   VITE_WEBHOOK_URL=https://hooks.zapier.com/hooks/catch/YOUR_WEBHOOK_ID/

SETUP STEPS:
1. Create a .env file in your project root
2. Add the VITE_WEBHOOK_URL variable with your webhook URL
3. Restart your development server
4. Test the contact form

The webhook will receive a JSON payload with:
- name: User's name
- email: User's email
- organization: User's organization (optional)
- message: User's message
- timestamp: When the message was sent
- source: "PV Automation Website"
